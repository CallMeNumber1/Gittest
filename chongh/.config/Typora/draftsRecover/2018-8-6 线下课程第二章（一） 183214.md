### EP02  循环数组

```c
#include<stdio.h>

int main(){
    int f[3] = {1, 0, 0}, sum = 0, n = 1;
    while(f[(n + 2) % 3] + f[(n + 3) % 3] <= 4000000){
        f[n % 3] = f[(n + 1) % 3] + f[(n + 2) % 3];
        if((f[n % 3] & 1) == 0) sum += f[n % 3];
        n += 1;
    }
    printf("%d\n",sum);
    return 0;
}
```

注意循环数组的初始化

初始化时n = 1, 求f[1], f[1]的前一项为f[0] = 1, f[1] 的前二项为f[2] = 0
